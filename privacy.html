üîë Permissions Explained

This extension requires the following permissions. Here's exactly why:

1. activeTab

Purpose: To detect and access video elements on the current active tab

Usage: Only when you manually activate Picture-in-Picture mode

Data Access: Limited to the active tab, only when you use the extension

Data Collection: None - we only detect video elements, not collect data

2. storage

Purpose: To save your custom keyboard shortcut preferences

Usage: To store and retrieve your settings locally

Data Stored: Only keyboard shortcut configurations (modifier keys + letter)

Location: Stored locally on your device

3. host_permissions: <all_urls>

Purpose: To enable Picture-in-Picture on any website with video content

Usage: To inject the content script that detects video elements

Data Access: Limited to video element detection only

Data Collection: None - no data is collected or transmitted from visited websites
